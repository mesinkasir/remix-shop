####  Install 
- `git clone https://github.com/mesinkasir/remix-shop.git`
- `cd remix-shop`
- rename .env.example to .env
- `npm install`
- `npx prisma db push`
- `npx prisma db seed`
- `npx prisma migrate dev`
- `npm run dev`
- localhost:3000
